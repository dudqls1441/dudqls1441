
데이터 통신

data
-데이터를 만들어 사용하는 사용자간 합의된 형식으로 된 사실, 개념, 명령 등을 표현한 것이다.
-컴퓨터에서는 데이터는 2진 디지털 정보 단위인 비트로 표현된다.

원거리 통신
-전화, 핸드폰, TV 등과 같이 먼거리에서 행해진느 통신을 의미한다.

데이터 통신의 기본 특성
전달 - 정확한 목적지에 전달, 원하느 장치나 사용자에게 전달한다.
정확성 - 더이터를 정확하게 전달한다.
적시성 - 데이터를 적정 시간 내에 전달한다.
파형 난조 - 음성이나 동영상 패킷 도착 시간이 서로 일정치 못하다.

프로토콜 -  두 장치간의 데이터 통신 규약
구성 요소
메시지 - 통신 대상인 전송되는 정보(데이터)
-정보는 문자,소리,그림,숫자,영상 또는 이들의 집합

송신자(sender)
-메시지를 보내는 장치 (컴퓨터, 핸드폰)

수신자(Receiver)
-메시지를 받는 장치

전송매체
-메시지가 송신자에서 수신자까지 메시지를 전달하는 물리적인 결로

프로토콜
-데이터 통신을 제어하는 규칙들의 집합

데이터 표현
문자 : 비트 패턴으로 구성되고 0과1로 된 비트들의 순차열로 표현
숫자 : 비트 패턴을 사용하여 표현
화성 : 픽셀(pixel),해상도(resolution)
음성 : 연속 신호
동영상 : 연속적인 개체 또는 여러 화상의 조합
